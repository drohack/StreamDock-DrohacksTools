How to set up

1. Go to discord.com/developers/applications and create a New Application.

2. On the OAuth2 page: copy the Client ID and Client Secret, and add redirect URL http://localhost.

3. Paste both here, click Save + Connect, then approve the popup inside Discord.

Rotate = chat (voice) volume. Press = deafen/undeafen.
